# Review Insights+ Frontend

Interface web connectee a l'API `Review Insights+`.

## Pages incluses

- `index.html` : presentation produit
- `app-online.html` : application web connectee au backend online
- `demo-api.html` : mini console de test

## Configuration

Le frontend pointe par defaut vers:

- `https://francescogiraldi-review-insights-api.hf.space`

L'utilisateur peut changer la base URL directement dans l'interface.
